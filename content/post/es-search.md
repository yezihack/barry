## 概要

Elasticsearch对外提供的API遵循REST原则

1. GET
2. POST
3. PUT
4. DELETE

## 数据操作

### 新增记录

```
curl -X PUT 'localhost:9200/accounts/person/1' -d '
{
  "user": "张三",
  "title": "工程师",
  "desc": "数据库管理"
}' 
```
## 查询
### 带条件查询

query: 查询

desc: 是指 json 数据里的字段, 即key

```
curl http://192.168.70.220:9200/accounts/person/_search -d '
{
  "query" : { "match" : { "user" : "李" }}
}'
```
